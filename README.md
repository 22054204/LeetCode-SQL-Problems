# LeetCode SQL Problems

A curated collection of my **LeetCode SQL** solutions written as part of my journey to strengthen SQL fundamentals, database querying skills, and interview preparation.

This repository serves as a structured archive of SQL problems solved on LeetCode. Each problem is organized in its own directory and includes the original problem statement along with the accepted SQL solution, making it easy to revisit concepts, review different query techniques, and track progress over time.

---

## Repository Overview

Each solved problem is stored in its own dedicated folder using the following structure:

```text
LeetCode-SQL-Problems/
└── {problem-number}-{problem-title}/
    ├── README.md
    └── solution.sql
```

Each folder contains:

* **README.md**

  * Original problem statement
  * Examples
  * Constraints

* **SQL Solution**

  * Accepted solution submitted on LeetCode
  * Preserved exactly as submitted

Keeping the problem statement and solution together makes every folder self-contained, allowing any problem to be understood without visiting LeetCode.

## Why LeetCode SQL?

LeetCode provides a practical way to apply SQL concepts through real-world database problems frequently asked in technical interviews.

Rather than practicing isolated queries, the problems require combining multiple SQL concepts to solve complete database scenarios, helping develop stronger analytical thinking and query-writing skills.

## Problem-Solving Approach

For each problem, I focus on understanding the database schema, relationships between tables, and the expected result before writing the query.

Whenever possible, I explore different approaches before arriving at the final accepted solution. The goal is not only to solve the problem but also to write queries that are clear, efficient, and easy to understand.
